循环的栅栏

只要满足第一个参数的条件，就可以只能第二个参数的操作；如果第一次只能没执行完，继续循环执行。
比如一共81个人，规定满20个人就要发车，那么20+20+20+20发四次车，剩下的一个人必须等待再来19个人，才能执行发车的动作
