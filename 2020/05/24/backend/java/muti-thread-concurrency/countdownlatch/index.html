门闩

作用是等待所有线程结束（countDown()/await()）。
当然也可以用join。
但是用Countdown更加灵活。比如可以在一个线程里面countDown()多次，在一个线程里面控制门闩，而join只能等待一个线程结束啦才能继续往后走
