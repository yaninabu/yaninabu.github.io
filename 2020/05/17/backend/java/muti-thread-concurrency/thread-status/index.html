创建时间:2020-05-17 15:39

线程的六种状态

  1. new MyThread()的时候进入New状态
  2. 执行start方法的时候进入Runnable状态；其中Runnable状态又分为Ready和Running状态，所谓Ready状态就是被线程调度器执行，在可执行的队列中，随时可以执行进入Running状态；Running状态是正在执行
  3. 线程执行完进入terminated终止状态，运行结束
  4. 线程运行中遇到Thread.sleep/o.wait/o.join/LockSupport.parkNanos/LockSupport.Until方法进入TimeWait状态。TimeWait状态等待时间结束，进入Runnable状态
  5. 运行中线程遇到o.wait/o.join/LockSupport.park方法进入wait状态，直到再遇到o.notify/o.notifyAll/LockSupport.unPark方法进入Runnable状态
  6. 当线程遇见等待同步锁进入Block状态，直到获得锁进入Runnable状态
